# Creative Connections Production Website

Domain: https://ccstreamteam.com

## Edit the site in one place

Open `assets/js/site-data.js` to change:
- social and Discord links
- creator names, bios, and Twitch links
- schedule
- events
- application form URL
- contact email
- homepage statistics

## Publish on GitHub Pages

1. Create a public GitHub repository named `creative-connections-website`.
2. Extract the ZIP and upload all extracted files and folders to the repository root.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.
6. In **Custom domain**, enter `ccstreamteam.com`.
7. Configure the WordPress.com DNS records as described in `LAUNCH-GUIDE.txt`.
8. After DNS verifies, enable **Enforce HTTPS**.

The `CNAME` file is already included.
